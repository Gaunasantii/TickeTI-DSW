# TickeTI - Help Desk

Es una plataforma de gestión de incidentes y requerimientos técnicos (Help Desk) diseñada para centralizar el soporte IT. Permite a los usuarios reportar fallas y a los administradores gestionar la resolución de manera eficiente, midiendo tiempos de respuesta y niveles de prioridad
